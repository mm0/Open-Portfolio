DB Auto-Backup


Role: Lead Developer, Database Schema Designer
Technologies: Ruby, MySQL, SSH, gzip
URL(s):
Github Repo

This project is a command-line utility for automating various DBA administration tasks such as creating and restoring backups locally, remotely, for all your servers and utilizes SSH for automation through Crons, etc.

Over the years as a system and database adminstrator, it became apparent that various DBA tasks would eventually be needed for a virtually any project one would work on. Those tasks usually have to do with backing up and restoring databases, setting up replication, modifying schema, etc.

These tasks can take a lot of time to accomplish for someone with not very much experience, so the goal was to create a simple utility that would allow to manage all these tasks via command-line with only a one-time configuration of Server IP Addresses, credentials and database types and names.
