Babeebook was launched in 2013 as a way for parents to store and share videos, images and posts about their children.

As lead backend developer for this project, it was my goal to setup an API for the iOS app to send and receive data between our hosted server. The project required for users to be able to create accounts, integrate with Facebook and Twitter APIs in order to upload media and share news about their children with authorized friends and followers. This app is similar to Instagram in how your social network is created and expanded although with the ability to upload and post more than just pictures.

This project used my API Builder Framework to seamlessly and near effortlessly create an organized and verifiable Web Application programming interface that handled media uploads and various account management features.
