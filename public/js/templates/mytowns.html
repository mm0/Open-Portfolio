MyTowns was launched in 2012 in collaboration with Fox CT as an App for fans of the news station to stay up-to-date on their local news. The App offered many features such as local reviews and personalized push notifications for local news, in addition to the ability to submit "Reports" to Staff including images, videos, location information, and other descriptions that could potentially become "Featured" reports within the App and even become featured on TV. The App also included the ability to send targeted Sponsored Push Notifications to users as well as Notifications regarding current hashtags and trends.

My role in this project was that of Lead Back End Developer. It was my responsibility to design the database schema for this project to store user information, preferences, locations data, etc and connect this to an external private API that the Mobile App itself would connect to. In addition to this, several features required an administration Panel for Staff of Fox CT to moderate Users Reports, set trending hashtags, deliver scheduled Sponsored Push Notifications and designate specific reports as featured. This project also featured a lot of XML feed parsing to create customized local feeds for users and also required efficient indexing and storing within our databases.

This project utilized the API Builder and Query Engine Frameworks to create an easy to use and easily manageable API, useable by both the Mobile App and Admnistrative Panel by authorized users.
